# 脚本语言概览

脚本语言模块收集了 Python 与 Shell 相关的实战笔记，侧重于自动化运维、批量任务和常见的系统工具脚本。

## 学习路径

1. **掌握基础语法**：了解变量、流程控制、函数与模块的基本写法。
2. **熟悉标准库与常用工具**：如 `os`、`subprocess`、`requests` 等 Python 模块，以及 Shell 中的管道、重定向、文本处理命令。
3. **结合实战案例练习**：参考本目录下的脚本示例，将脚本嵌入 CI/CD、巡检、告警等场景。

## 内容分类

- `python/`：Python 语言特性、标准库用法以及自动化脚本案例。
- `shell/`：Shell 基础语法、脚本技巧和常见排障脚本。

如果你是第一次进入本章节，推荐先阅读 Python、Shell 的入门文档，再根据需求查阅对应的专题内容。
